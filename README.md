Mini Project Management System 

A professional, lightweight, and fully functional Project Management Tool built as part of the. This application allows users to manage projects and their associated tasks with features like real-time updates, filtering, and server-side pagination.

Features

Project Management: Create, view, and delete projects.
Task Management: Add tasks to specific projects with Title, Description, Priority, and Due Date.
Inline Editing: Professional inline update feature to edit Task details, Priority, and Status without leaving the page.
Dynamic Filtering: Filter tasks based on their status (To-Do, In-Progress, Done).
Server-side Pagination: Optimized data loading for projects to handle large datasets efficiently.
Responsive UI: Clean and professional design using Vanilla CSS (No frameworks like Tailwind for a lightweight footprint).


Tech Stack

Frontend:
- HTML5 & CSS3 (Custom Styling)
- Vanilla JavaScript (ES6+)
- Fetch API for asynchronous calls

Backend:
- Node.js & Express.js
- MongoDB Atlas (Cloud Database)
- Mongoose (ODM)


 Prerequisites

Before you begin, ensure you have the following installed:
- [Node.js](https://nodejs.org/) (v14 or higher)
- [MongoDB](https://www.mongodb.com/) account (Atlas) or Local Compass.
